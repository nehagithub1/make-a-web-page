Lesson 3: Adding CSS Style to HTML Structure

Styling on a Page

How does browser know what style to apply

HTML is language of the web with syntax and rules

Written in a document

But with their rules

Word in the language is TAG

Browser turns HTML TAG into elements that form a tree

Can do that bec of DOM (Document Object Model)

DOM tree that browser built from HTML Doc

Each HTML tag creates an element in DOM that the browser uses to display the page

An element is created from the starting tag to ending tag and everything in between is content

Tag describes elements (para/image/division/attributes with values)

CSS

Allows you to specify styles such as color

CSS – can add color as color name or rgb color

CSS is code written to control the "style" of HTML elements.
